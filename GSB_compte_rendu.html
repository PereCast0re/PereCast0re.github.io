Appli Compte-Rendu


Présentation :

L’entreprise GSB est une entreprise pharmaceutique engageant des visiteurs pour faire la promotion de leurs médicaments.
À la fin de chaque opération, les visiteurs doivent remplir un compte rendu dans lequel ils décrivent les médicaments qui ont été présentés, les échantillons qui ont été offerts et s’ils ont donné une documentation ou non.

L’entreprise veut désormais que les comptes rendus soient numérisés pour permettre une meilleure organisation, pour cela, elle demande de mettre en place une application en ligne utilisant PHP pour la gestion de la base de données.

Répartitions du Projet :

Le projet à était réaliser avec l'architecture MVC.
Pour respecter les codes du model d'architecture MVC, on as répartie les fonctionnalités en différents controlleur.



Controlleur Compte-Rendu :


  • c_ajouterCompteRendu.php
  • c_modifierCompteRendu.php
  • c_consulterCompteRenduPeriode.php
  • c_consulterCompteRenduRegion.php
  • c_consulterCompteRenduSecteur.php
  • c_consulterCompteRenduVisiteur.php

Controlleur pour les autres fonctionnalités


  • c_connexion.php
  • c_ajouterPraticien.php
